 *Правила заполнения объектов _для сборки:_*
:one: Сообщение должно начинаться c _--obj {тикет Jira|ссылка на тикет}_
:two: Вторая строка - название проекта, для сборки.
Например название релиза или специфическая метка(18Q4, e2e и т.д.)
(Название проектов можно получить по команде _--projects_)
:three:Объекты передаются в формате _{Тип объекта}:{Название 1},{Название 2} .. {Название N}_
При корректном завершении операции бот отвечает *done*
*Пример:*
```
 --obj OTP-1234
 18Q4
 BC:Contact
 Applet:Contact Assoc Applet,Contact Attachment Applet
```


 *Правила заполнения объектов _для компиляции:_*
:one: Сообщение должно начинаться c _--comp_
:two: Объекты передаются в формате _{Тип объекта}:{Название 1},{Название 2} .. {Название N}_
При завершении операции бот отвечает *успешно или не успешно* выполнил процедуру
*Пример:*
```
--comp
 BC:Contact
 Applet:Contact Assoc Applet,Contact Attachment Applet
```
